help_outline How to Use Distance to Shannon Calculator

pin_drop Getting Ship Position

From Marine Traffic:

  1. Find your vessel on Marine Traffic
  2. Right-click on the vessel icon
  3. Select "Copy Coordinates" from the menu
  4. Paste the coordinates directly into the position input box

Marine Traffic "Copy Coordinates" feature

Manual Entry:

You can also enter the position manually by filling in the Latitude and Longitude fields (degrees and minutes).

speed Setting Ship Speed

Important: Remember to set the vessel's speed in knots. The calculator uses this to estimate arrival times.

route Waypoint Selection

The calculator automatically selects the next waypoint based on the ship's position. A red line shows the route from the ship's position to the next waypoint.

Override Waypoint Selection:

If the automatic waypoint selection looks incorrect, simply click on any waypoint marker on the map to manually select it as the next waypoint. The route will update immediately.

schedule Time Delay

Use the "Delay" fields (hours and minutes) if you want to calculate from a time in the past. The "From" time in the results will reflect this adjustment.

save Saving Calculations

To save your calculation for later reference:

  1. Enter a Ship Name in the input field
  2. Click the Save button
  3. Your calculation will appear in the "Recent Calculations" table below

Note: Saving is optional - you can use the calculator without saving.

info_outline Understanding Results

  • From: Calculation time (current time minus any delay)
  • Next Waypoint: The selected navigation waypoint
  • Dist to WP: Distance to the next waypoint in nautical miles
  • Dist to Scattery: Total distance to Scattery Island
  • Time to Scattery: Estimated travel time in hours
  • ETA Kilcredaun: Estimated time of arrival at Kilcredaun Point
  • ETA Scattery: Estimated time of arrival at Scattery Island
