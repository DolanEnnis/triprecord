How to use this page

This page shows a list of recent trips.

Click on a row to see more details or perform actions.

  • Rows in gray are confirmed trips.
  • Confirmed trips you have created can be edited by clicking on them.
  • Rows in white are trips that have not been confirmed yet.

The button allows you to create a new trip record from scratch.

The warnings in the note column are only advisory!

However check your trips for duplicates, spelling mistakes, or missing trips.

You can filter them using the controls at the top.

  • Filter Trips: Search for trips by ship, port, pilot, etc.
  • All/In/Out: Filter trips by direction.
  • All Trips/My Trips: Show all trips or only your own (and unassigned trips).

The button allows you to export the currently visible confirmed trips to a CSV file.
